Latest Projects

Bankist

A simple Bank App built with HTML, CSS, and JavaScript that lets users log in, view transactions, request loans with interest, close accounts, and auto-logout after inactivity — a hands-on project that strengthened my JavaScript and problem-solving skills.

Online-Shop

This project is a responsive e-commerce frontend built with React and Tailwind CSS, featuring a dynamic product search, real-time cart updates, and a mobile-friendly navigation bar with a hamburger menu. It uses Redux Toolkit for state management and React Router for seamless navigation, delivering a modern and user-focused shopping experience while demonstrating clean, scalable, and maintainable frontend architecture.

Task Manager

A Task Manager built with HTML, CSS, and JavaScript provides a seamless way to organize and track tasks efficiently. With features like task creation, editing, and completion tracking, it enhances productivity and user experience. The intuitive interface ensures ease of use, while JavaScript enables dynamic interactions for smooth functionality. Designed to be responsive and user-friendly, this task manager helps streamline workflow and improve task management.

Weather-App

This weather app is built with React, JavaScript, and CSS. It allows users to search for any city and instantly see real-time weather details such as temperature, humidity, and wind speed. The interface features a modern, responsive design with smooth animations and a clean user experience. It demonstrates strong frontend skills in working with APIs, managing React components, and creating visually appealing layouts with CSS.
